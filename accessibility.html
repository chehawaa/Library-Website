Accessibility & Inclusivity

We are committed to ensuring our digital library is accessible to everyone, regardless of age, ability, or background. Below are some of the key features that support an inclusive user experience:

  • Screen-reader Compatibility: Fully optimized for users relying on screen-reading technology.
  • Alt Text for Images: Descriptive text is provided for all images to ensure visual content is accessible to all.
  • Adjustable Font Size & Contrast: Users can change text size and enable high-contrast display for better readability.
  • Keyboard-only Navigation: The entire website can be navigated without a mouse for improved accessibility.
  • Inclusive & Diverse Content: We provide culturally respectful, gender-neutral, and multilingual content.
  • Mobile-friendly Design: The website adapts seamlessly across devices, ensuring consistent access anywhere.
Our Pledge: We continuously improve accessibility through user feedback, technology updates, and collaboration with accessibility experts. If you encounter barriers while using our platform, please let us know so we can make it better for everyone.
